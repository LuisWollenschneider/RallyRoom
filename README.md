# Rally Room

A collection of Tennis, Beach Tennis, Pickleball and Padel drills, games and strategies.

---

Site available at: https://luiswollenschneider.github.io/RallyRoom/

## Workflow

1. Create a new entry on https://luiswollenschneider.github.io/RallyRoom/editor/

2. Set all settings in the sidebar

3. Configure the court diagram

4. Write the description in Markdown

5. Check the preview to make sure everything looks good

6. Click ".md Download" to download the generated Markdown file

7. Clone this repository: `git clone https://github.com/LuisWollenschneider/RallyRoom.git`

8. Move the downloaded Markdown file into the `content/<sport>/` folder of the cloned repository

9. Commit and push the changes
```sh
git add content/
git commit -m "add: My Drill Name"
git push
```
